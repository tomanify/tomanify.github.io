درباره Tomanify و این سایت راهنما

اگر حس کرده‌اید اطلاعات افزونه بین پنل وردپرس، WordPress.org و توضیح‌های پراکنده پخش شده، این صفحه برای همین ساخته شده است: یک مرجع فارسی واحد، روشن و قابل پیگیری.

این سایت فقط ترجمهٔ متن‌های فنی نیست؛ هدفش این است که مسیر استفاده، تصمیم‌گیری و عیب‌یابی را برای کاربر واقعی ساده‌تر کند.

سه جایی که باید Tomanify را بشناسید

  1. WordPress.org برای معرفی رسمی افزونه و انتشار نسخه‌ها
  2. راهنمای داخل پنل افزونه برای شروع سریع و Tooltipها
  3. همین سایت برای توضیح فارسی، مثال‌ها، منابع نرخ، FAQ و راه‌حل‌های عملی

چرا Tooltip مهم است؟

Tooltip داخل افزونه سریع‌ترین راه برای درک نقش تب‌هاست: Dashboard، General، Sources، Advanced، Shortcode، Products، Backup و About. این بخش برای لحظه‌ای ساخته شده که تازه افزونه را باز کرده‌اید و می‌خواهید بدون آزمون و خطا بفهمید از کجا باید شروع کنید.

در این سایت، همان منطق را با مثال، تصویر و توضیح‌های تکمیلی باز کرده‌ایم.

اسکرین‌شات صفحه درباره

پیام اصلی این سایت

Tomanify یک افزونهٔ رایگان و متن‌باز است، اما استفادهٔ حرفه‌ای از آن نیاز به توضیح دقیق دارد: انتخاب منبع نرخ، درک Premium و Margin، و فهم این‌که چه وقت منبع رایگان کافی است و چه وقت باید سراغ API شخصی‌تر رفت.

این سایت برای همین سطح از شفافیت ساخته شده است.

سوالات متداول این صفحه

جواب کوتاه به سوال‌هایی که معمولاً قبل از نصب، هنگام تنظیمات، یا زمان عیب‌یابی پیش می‌آید.

راهنمای اصلی Tomanify را از کجا پیدا کنم؟

سه مرجع اصلی وجود دارد: صفحه WordPress.org، راهنمای داخل پنل افزونه، و همین سایت که نسخهٔ فارسی و متمرکز راهنماها را نگه می‌دارد.

Tooltip یا تور داخل افزونه چه کمکی می‌کند؟

برای فهم سریع نقش هر تب و مسیرهای اصلی مناسب است. اگر نمی‌خواهید همه مستندات را یک‌جا بخوانید، Tooltip سریع‌ترین راه شروع است.

آیا این سایت فقط برای برنامه‌نویس‌هاست؟

خیر. بخش مهمی از بازنویسی جدید سایت برای این بوده که محتوای کاربرپسند، تصمیم‌محور و قابل استفاده برای صاحب فروشگاه هم اضافه شود.

چرا این سایت با وجود رایگان بودن افزونه مهم است؟

چون پراکندگی اطلاعات را کمتر می‌کند و یک مرجع فارسی واحد برای بروزرسانی‌ها، توضیح تنظیمات و روش‌های کار در اختیار کاربر می‌گذارد.
