تنظیمات پیشرفته

این صفحه برای زمانی است که می‌خواهید رفتار افزونه را دقیق‌تر کنترل کنید: کاهش نوسان، کار در شرایط آفلاین، یا انتقال تنظیمات بین سایت‌ها.

۱) Threshold (آستانه تغییر نرخ)

Threshold درصد حداقل تغییر لازم برای ذخیره نرخ جدید است. اگر تغییرات کمتر از این مقدار باشد، افزونه نرخ جدید را ذخیره نمی‌کند و سراغ آپدیت محصولات نمی‌رود.

  • پیشنهاد عمومی: ۰٫۱٪ تا ۰٫۵٪
  • برای جلوگیری از نوسان زیاد و آپدیت‌های ریز: مقدار را بالاتر ببرید.
  • برای آپدیت همیشه: مقدار را ۰ بگذارید (ممکن است فشار روی هاست بیشتر شود).

۲) Premium (تبدیل نرخ رسمی به نرخ آزاد)

وقتی Source Mode روی CBI باشد، افزونه نرخ رسمی بانک مرکزی را می‌خواند و با ضریب Premium آن را به «نرخ آزاد تخمینی» تبدیل می‌کند.

  • اگر برای USD لینک بانک مرکزی را تنظیم کرده باشید و منبع JSON هم داشته باشید، افزونه می‌تواند Premium را به‌صورت تطبیقی (EMA) به‌روزرسانی کند.
  • اگر اینترنت محدود است یا JSON در دسترس نیست، Premium را دستی تنظیم کنید تا خروجی نزدیک‌تر به بازار آزاد باشد.

۳) Offline Mode (برآورد آفلاین)

در حالت Offline افزونه هیچ درخواست خارجی نمی‌زند و نرخ‌ها را بر اساس یک «مرجع ذخیره‌شده» و نرخ رشد سالانه (CAGR) تخمین می‌زند:

  • Reference Snapshot: آخرین نرخ رسمی USD (به ریال) + زمان ذخیره.
  • CAGR: نرخ رشد سالانه برای برآورد روند (مثلاً ۰٫۲۵ یعنی ۲۵٪ در سال).
نکته: Offline یک مدل تخمینی است؛ برای قیمت‌گذاری دقیق همیشه JSON بازار آزاد یا بانک مرکزی (با Premium) گزینه‌های بهتری هستند.

۴) گرد کردن و «پایه تومان»

  • برای جلوگیری از تبدیل دوباره، افزونه مقدار tomanify_price_base_tmn را به‌عنوان «پایه تومان» ذخیره می‌کند.
  • قیمت نهایی برای ووکامرس به‌صورت پیش‌فرض به نزدیک‌ترین ۱۰۰۰ گرد می‌شود (برای ظاهر قیمت‌گذاری بهتر).
  • تغییر واحد تومان/ریال باعث دریافت دوباره نرخ نمی‌شود؛ فقط قیمت ووکامرس از روی پایه تومان بازنویسی می‌شود.

مطالعه بیشتر

  • منابع نرخ (Source Mode و لینک‌ها)
  • بکاپ و انتقال (Export/Import)
  • امنیت و حریم خصوصی

سوالات متداول این صفحه:

ما سوالات پرتکرار را که ممکن است به درک بهتر این صفحه کمک کنند را در اینجا قرار داده ایم.

Threshold را روی چه عددی بگذارم؟

برای اکثر فروشگاه‌ها ۰٫۱٪ تا ۰٫۵٪ مناسب است. اگر محصولات زیاد دارید یا نوسان‌های ریز برایتان مهم نیست، عدد بالاتر کمک می‌کند.

Offline دقیقاً چقدر دقیق است؟

Offline یک مدل تخمینی است و دقت آن به آخرین Reference Snapshot و تنظیم CAGR/Premium بستگی دارد. برای دقت بالا از JSON یا CBI استفاده کنید.

Premium را دستی تنظیم کنم یا خودکار؟

اگر هم JSON و هم URL رسمی USD بانک مرکزی را دارید، حالت خودکار می‌تواند Premium را نزدیک‌تر کند. در غیر این صورت دستی تنظیم کنید.
