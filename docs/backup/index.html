بکاپ و انتقال تنظیمات Tomanify

اگر سایت تست و سایت اصلی دارید یا می‌خواهید تنظیمات و محصولات foreign را بین دو محیط جابه‌جا کنید، این صفحه برای شماست.

چه زمانی Export/Import مفید است؟

وقتی در سایت تست همه چیز را تنظیم کرده‌اید و نمی‌خواهید در سایت اصلی دوباره همان URLها، تنظیمات، و محصولات foreign را از اول وارد کنید.

چه چیزی را باید بعد از Import بررسی کنید؟

چند محصول نمونه، منبع نرخ، واحد تومان/ریال و نتیجهٔ Update Rates. مخصوصاً اگر سایت‌ها از نظر قالب، کش یا موجودی محصول متفاوت باشند.

یک روال امن برای انتقال

  1. در سایت تست همه تنظیمات را کامل کنید.
  2. Export بگیرید و فایل را نگه دارید.
  3. در سایت مقصد import کنید.
  4. نرخ‌ها را بروزرسانی کنید و چند محصول را تست بگیرید.
  5. اگر کش فعال است، کش را پاک کنید.

تصویر: حذف داده‌ها هنگام حذف افزونه

این گزینه برای چه زمانی مهم است؟

اگر روی سایت تست زیاد نصب و حذف انجام می‌دهید، نگه داشتن داده‌های محصولات foreign می‌تواند زمان راه‌اندازی دوباره را کم کند. اما اگر می‌خواهید افزونه را کاملاً پاک کنید، باید گزینه حذف کامل داده‌ها را انتخاب کنید.

سوالات متداول این صفحه

جواب کوتاه به سوال‌هایی که معمولاً قبل از نصب، هنگام تنظیمات، یا زمان عیب‌یابی پیش می‌آید.

Export دقیقاً چه کمکی می‌کند؟

برای زمانی که می‌خواهید تنظیمات و محصولات foreign را از سایت تست به سایت اصلی یا برعکس منتقل کنید و دوباره همه چیز را دستی نسازید.

آیا بعد از Import باید چیزی را دوباره چک کنم؟

بله، بهتر است منابع نرخ، نرخ‌های ذخیره‌شده و چند محصول نمونه را بعد از import بررسی کنید.

اگر نام یا SKU محصول‌ها فرق کند چه می‌شود؟

تطبیق محصولات در import باید با دقت انجام شود؛ بنابراین قبل از انتقال گسترده، روی چند محصول نمونه تست بگیرید.

بکاپ جایگزین منبع نرخ زنده است؟

خیر. بکاپ برای انتقال تنظیمات و متادیتاست؛ منبع نرخ زنده و اعتبار آن باید جداگانه درست تنظیم شود.
