بکاپ و انتقال (Export/Import)

برای انتقال تنظیمات بین سایت تست و اصلی، یا نگهداری نسخه پشتیبان قبل از تغییرات مهم، از بکاپ JSON استفاده کنید.

Export چه چیزهایی را ذخیره می‌کند؟

  • Settings: گزینه‌های اصلی مثل واحد (تومان/ریال)، زبان افزونه، زمان‌بندی Cron، Source Mode، Threshold، Premium و تنظیمات شورتکد.
  • Links: لینک‌های منبع (JSON و RSS/XML بانک مرکزی) که شما وارد کرده‌اید.
  • Products: فقط محصولات foreign (انتشار یافته) شامل metaهای: foreign، currency، price، margin و در صورت وجود tomanify_price_base_tmn.
یادآوری: فایل بکاپ شامل اطلاعات سفارش‌ها/کاربران/پرداخت‌ها نیست؛ فقط داده‌های مربوط به Tomanify را نگه می‌دارد.

روش Export

  1. در پیشخوان وردپرس به Tomanify → Backup بروید.
  2. تعیین کنید چه مواردی داخل بکاپ باشد (Settings/Links/Products).
  3. روی Download Backup بزنید تا فایل JSON دانلود شود.

روش Import

  1. در همان تب Import Backup فایل JSON را آپلود کنید.
  2. افزونه فایل را اعتبارسنجی می‌کند و سپس تنظیمات/لینک‌ها/محصولات را بازیابی می‌کند.
  3. برای محصولات، بازیابی ابتدا با SKU انجام می‌شود و اگر SKU نبود/پیدا نشد، با عنوان دقیق محصول تلاش می‌کند.
  4. بعد از Import، محاسبه قیمت‌ها در صف قرار می‌گیرد تا قیمت ووکامرس بر اساس داده‌های بازیابی‌شده بروزرسانی شود.

نکات مهم

  • قبل از Import روی سایت اصلی، بهتر است ابتدا روی سایت تست انجام دهید.
  • اگر عنوان محصولات متفاوت است یا SKU ندارید، ممکن است برخی محصولات match نشوند.
  • برای امنیت، فایل بکاپ را مثل یک فایل تنظیمات حساس نگه دارید (به‌خصوص اگر داخل آن URL خصوصی دارید).

سوالات متداول این صفحه:

ما سوالات پرتکرار را که ممکن است به درک بهتر این صفحه کمک کنند را در اینجا قرار داده ایم.

بکاپ دقیقاً کجا ذخیره می‌شود؟

بکاپ به‌صورت فایل JSON دانلود می‌شود و روی سرور شما ذخیره دائمی نمی‌گردد (مگر اینکه خودتان آن را نگه دارید).

Import چه چیزی را overwrite می‌کند؟

گزینه‌های whitelist شده و metaهای محصولات match‌شده بروزرسانی می‌شوند. سپس محاسبه قیمت‌ها در صف قرار می‌گیرد.

چطور مطمئن شوم محصولات درست match می‌شوند؟

بهترین روش داشتن SKU یکتا برای محصولات است. اگر SKU ندارید، عنوان دقیق محصول معیار جایگزین است.
