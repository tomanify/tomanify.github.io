سوالات پرتکرار

سوال‌هایی که معمولاً در راه‌اندازی و استفاده از Tomanify پیش می‌آید — با جواب کوتاه و دقیق.

آیا Tomanify به‌صورت پیش‌فرض لینک منبع نرخ را وارد می‌کند؟

خیر. تمام URLها خالی هستند و درخواست خارجی فقط پس از ورود URL توسط مدیر انجام می‌شود.

اگر نرخ‌ها کمی تغییر کنند ولی محصولات آپدیت نشوند، مشکل چیست؟

گزینه Change threshold % را بررسی کنید. تغییرهای کوچک زیر آستانه برای کاهش نویز نادیده گرفته می‌شوند.

آیا می‌توانم فقط نرخ دستی وارد کنم؟

بله. Source Mode را روی Manual بگذارید و عدد تومان هر ارز را وارد کنید.

ارزهای پشتیبانی‌شده کدام‌اند؟

USD، EUR، AED، TRY، CNY

آیا این افزونه اطلاعات مشتری/سفارش را به بیرون ارسال می‌کند؟

خیر. درخواست‌های بیرونی فقط برای دریافت نرخ (از URLهایی که شما وارد می‌کنید) است و هیچ داده شخصی مشتری ارسال نمی‌شود.
