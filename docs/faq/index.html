سوالات پرتکرار درباره Tomanify

اگر می‌خواهید قبل از ورود به صفحه‌های عمیق‌تر، جواب پرسش‌های رایج را یک‌جا ببینید، این صفحه برای شماست.

نرخ آزاد یا رسمی؟

برای بیشتر فروشگاه‌ها، هدف اصلی نرخ بازار آزاد است. اما اگر لازم باشد، فیدهای رسمی بانک مرکزی هم می‌توانند به‌عنوان پشتیبان تنظیم شوند و با Premium به نرخ آزاد نزدیک‌تر شوند.

API پولی یا رایگان؟

در خیلی از سناریوها منبع رایگان کافی است. تفاوت اصلی زمانی مهم می‌شود که کسب‌وکار شما روی سرعت بروزرسانی و نوسان‌های خیلی کوچک حساس باشد.

FAQ اصلی Tomanify

جواب کوتاه به سوال‌هایی که معمولاً قبل از نصب، هنگام تنظیمات، یا زمان عیب‌یابی پیش می‌آید.

آیا Tomanify به‌صورت پیش‌فرض URL منبع نرخ را پر می‌کند؟

خیر. فیلدهای URL به‌صورت پیش‌فرض خالی هستند و فقط پس از ورود آدرس توسط مدیر، درخواست خارجی انجام می‌شود.

آیا برای استفاده از Tomanify حتماً باید API بخریم؟

برای اکثر فروشگاه‌ها خیر. می‌توانید با JSON رایگان بازار آزاد شروع کنید و در صورت نیاز، Margin و منابع پشتیبان را تنظیم کنید.

آیا افزونه نرخ بازار آزاد را می‌فهمد یا فقط نرخ رسمی؟

تمرکز اصلی روی نرخ بازار آزاد است، اما در صورت نیاز می‌تواند از فیدهای رسمی بانک مرکزی هم استفاده کند و با Premium آن‌ها را به نرخ آزاد نزدیک‌تر کند.

اگر اینترنت محدود یا قطع شود چه می‌شود؟

می‌توانید فیدهای بانک مرکزی، Offline Mode و نرخ دستی را به‌عنوان پشتیبان تنظیم کنید تا فروشگاه فقط به یک منبع وابسته نماند.

چرا مرتب‌سازی قیمت در فروشگاه درست می‌ماند؟

چون افزونه قیمت نهایی ووکامرس را بازنویسی می‌کند و صرفاً یک نرخ نمایشی روی فرانت‌اند نشان نمی‌دهد.

آیا داده‌های مشتری یا سفارش به بیرون فرستاده می‌شود؟

خیر. درخواست‌های خارجی فقط برای گرفتن نرخ از URLهایی است که مدیر وارد می‌کند و اطلاعات شخصی مشتری ارسال نمی‌شود.

اگر فروشگاه من خیلی حساس است بهترین کار چیست؟

یا Margin محافظه‌کارانه بگذارید، یا از API اختصاصی‌تر استفاده کنید. منبع رایگان برای اکثر فروشگاه‌ها خوب است ولی نه لزوماً برای سناریوهای بسیار حساس.
