۱) بروزرسانی خودکار قیمت محصولات

اگر قیمت خرید شما دلاری، یورویی، درهمی، لیر یا یوانی است، Tomanify می‌تواند آن را به تومان یا ریال تبدیل کند و قیمت نهایی ووکامرس را بر همان اساس بازنویسی کند. این فقط یک نمایش ظاهری نیست؛ خود فیلد قیمت ووکامرس هم آپدیت می‌شود.

بیشتر درباره بروزرسانی خودکار

۲) منابع نرخ قابل تنظیم

می‌توانید از JSON بازار آزاد، فیدهای رسمی RSS/XML، نرخ دستی یا مدل آفلاین استفاده کنید. یعنی مجبور نیستید به یک منبع خاص بچسبید و برای هر فروشگاه می‌توانید استراتژی مناسب‌تری بسازید.

منابع نرخ

۳) پاک‌سازی هوشمند کش

اگر قیمت محصول عوض شد اما سایت شما کش دارد، لازم نیست یک افزونه جدا برای purge نصب کنید. می‌توانید تعیین کنید بعد از بروزرسانی قیمت، فقط همان URLهای مرتبط دوباره تازه شوند تا کاربر سریع‌تر قیمت جدید را ببیند.

پاک‌سازی کش

۴) زبان مستقل افزونه

ممکن است پنل وردپرس شما انگلیسی باشد ولی بخواهید فقط Tomanify فارسی باشد؛ یا برعکس. زبان داخلی افزونه مستقل از زبان کل سایت کنترل می‌شود و این برای تیم‌های چندزبانه خیلی مفید است.

زبان افزونه

۵) ریست تنظیمات و حذف قابل کنترل

اگر تنظیمات به‌هم ریخت، می‌توانید افزونه را به حالت پیش‌فرض برگردانید. هنگام حذف هم می‌توانید تصمیم بگیرید همه داده‌ها کامل پاک شوند یا فقط تنظیمات حذف شود و داده‌های مهم محصولات برای استفاده بعدی باقی بمانند.

ریست و حذف

۶) بکاپ، بازیابی و انتقال

برای انتقال تنظیمات بین سایت تست و سایت اصلی، یا برای این‌که قبل از تغییرات مهم خیالتان راحت باشد، می‌توانید از تنظیمات خروجی بگیرید و بعداً آن را بازیابی کنید.

بکاپ و انتقال

این قابلیت‌ها برای چه جستجوهایی به درد می‌خورند؟

اگر کاربر با عباراتی مثل «چگونه قیمت‌های سایتم را خودکار کنم؟»، «آیا می‌شود قیمت فروشگاه اینترنتی را خودکار تغییر داد؟»، «آموزش بروزرسانی خودکار قیمت‌های وردپرس» یا «تبدیل قیمت اجناس از دلار به تومان در ووکامرس» جستجو کند، جواب همهٔ این‌ها در عمل به همین مجموعه قابلیت‌ها برمی‌گردد: منبع نرخ، محصول خارجی، پایه تومان، بروزرسانی زمان‌بندی‌شده، purge کش و تنظیمات امن.

سوالات متداول این صفحه

جواب کوتاه به سوال‌هایی که معمولاً قبل از نصب، هنگام تنظیمات، یا زمان عیب‌یابی پیش می‌آید.

Tomanify فقط نرخ ارز را نمایش می‌دهد یا قیمت ووکامرس را هم عوض می‌کند؟

فقط نمایش نرخ نیست؛ قیمت نهایی ووکامرس را هم از روی پایه تومان بازنویسی می‌کند تا مرتب‌سازی، فیلتر و گزارش‌ها درست بمانند.

آیا می‌توانم بدون افزونه کش جداگانه، بعد از آپدیت قیمت فقط همان صفحه محصول را تازه کنم؟

بله، قابلیت purge اختیاری دقیقاً برای همین سناریو ساخته شده تا فقط URLهای همان سایت شما دوباره پاک‌سازی شوند.

اگر زبان وردپرس انگلیسی باشد، می‌توانم فقط خود Tomanify را فارسی ببینم؟

بله، زبان افزونه از زبان کلی سایت مستقل است و می‌توانید فقط برای Tomanify زبان فارسی یا انگلیسی انتخاب کنید.

اگر تنظیمات افزونه به‌هم بریزد چه کنم؟

می‌توانید از Factory Reset استفاده کنید یا قبل از آن از تنظیمات خروجی JSON بگیرید و بعداً دوباره درون‌ریزی کنید.

هنگام حذف افزونه، همه اطلاعات پاک می‌شوند؟

این موضوع قابل تنظیم است؛ می‌توانید حذف کامل داشته باشید یا حذف سبک تا داده‌های پرچم محصول خارجی، ارز، قیمت و Margin برای استفاده بعدی باقی بمانند.
