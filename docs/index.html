تصویر: محصول خارجی

سوالات متداول این صفحه:

ما سوالات پرتکرار را که ممکن است به درک بهتر این صفحه کمک کنند را در اینجا قرار داده ایم.

از کجا شروع کنم؟

اگر تازه نصب کردید، از «شروع سریع» و بعد «نصب» و «منابع نرخ» شروع کنید.

اگر مشکل دارم کجا بروم؟

اول «رفع اشکال» را ببینید. برای تنظیمات حساس مثل Threshold و Offline هم صفحه «پیشرفته» را بخوانید.

برای انتقال تنظیمات بین سایت تست و اصلی چه کنم؟

صفحه «بکاپ» را ببینید؛ خروجی JSON بگیرید و روی سایت مقصد Import کنید.

آیا افزونه URL پیش‌فرض دارد یا داده‌ای ارسال می‌کند؟

خیر. برای توضیح کامل، صفحه «امنیت و حریم خصوصی» را ببینید.
