راهنمای فارسی Tomanify

این صفحه برای این ساخته شده که مسیر درست را سریع پیدا کنید؛ نه فقط بر اساس اسم تب‌ها، بلکه بر اساس کاری که واقعاً می‌خواهید انجام دهید.

راه‌اندازی در ۵ دقیقه
API رایگان بازار آزاد
بانک مرکزی به‌عنوان پشتیبان
محصول وارداتی و پایه تومان
پاک‌سازی کش و زبان مستقل
ریست، حذف و رفع اشکال

اگر تازه شروع کرده‌اید

برای نصب، وارد کردن JSON، تست اولین بروزرسانی و تنظیم اولین محصول خارجی.

شروع سریع نصب

اگر دنبال API رایگان هستید

برای فهمیدن اینکه آیا اصلاً لازم است API بخرید، لینک آمادهٔ JSON چیست، و چه زمانی Margin لازم می‌شود.

API رایگان منابع نرخ

اگر می‌خواهید قیمت‌ها خودکار عوض شوند

برای فهمیدن منطق بروزرسانی خودکار، ذخیره قیمت واقعی ووکامرس و این‌که چرا فقط نمایش نرخ کافی نیست.

بروزرسانی خودکار محصولات خارجی

اگر می‌خواهید همه قابلیت‌ها را یکجا ببینید

برای مرور کلی ویژگی‌های مهم افزونه از purge کش تا زبان مستقل، بکاپ، reset و حذف سبک یا کامل.

مرور قابلیت‌ها

اگر درباره کش، purge یا نسخه قدیمی قیمت مشکل دارید

برای وقتی که نرخ و قیمت در پنل عوض شده ولی کاربر هنوز نسخه قدیمی را می‌بیند.

پاک‌سازی کش رفع اشکال

اگر درباره زبان، reset یا حذف افزونه سوال دارید

برای وقتی که می‌خواهید فقط خود افزونه فارسی/انگلیسی باشد، تنظیمات را به حالت اول برگردانید، یا افزونه را سبک یا کامل حذف کنید.

زبان افزونه ریست و حذف

تصویر: متاباکس محصول خارجی

سوالات متداول این صفحه

جواب کوتاه به سوال‌هایی که معمولاً قبل از نصب، هنگام تنظیمات، یا زمان عیب‌یابی پیش می‌آید.

اگر تازه نصب کرده‌ام از کجا شروع کنم؟

اول شروع سریع را بخوانید، بعد نصب و منابع نرخ را تنظیم کنید. بعد از آن صفحه محصولات خارجی و در صورت نیاز شورتکد را ببینید.

اگر نمی‌خواهم API پولی بخرم کدام صفحه به درد من می‌خورد؟

صفحه API رایگان و سپس صفحه منابع نرخ را بخوانید. آن‌جا هم لینک آمادهٔ JSON را می‌بینید، هم روش پشتیبان بانک مرکزی را.

اگر می‌خواهم قیمت‌های فروشگاهم خودکار عوض شوند کدام صفحه مهم‌تر است؟

صفحه بروزرسانی خودکار قیمت و سپس صفحات محصولات خارجی، منابع نرخ و پیشرفته بهترین مسیر را نشان می‌دهند.

اگر می‌خواهم فقط خود Tomanify فارسی باشد کجا را ببینم؟

صفحه زبان افزونه را ببینید؛ آنجا منطق زبان مستقل افزونه از زبان کلی سایت توضیح داده شده است.

اگر بعد از آپدیت قیمت، کاربر هنوز قیمت قدیمی را می‌بیند چه کنم؟

صفحه پاک‌سازی کش و بعد رفع اشکال را ببینید؛ معمولاً مشکل از cache page یا CDN است.
