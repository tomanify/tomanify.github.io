راهنمای فارسی Tomanify

این صفحه برای این ساخته شده که مسیر درست را سریع پیدا کنید؛ نه فقط بر اساس نام تب‌ها، بلکه بر اساس کاری که واقعاً می‌خواهید انجام دهید.

راه‌اندازی در ۵ دقیقه
API رایگان بازار آزاد
بروزرسانی دستی و کرون
بانک مرکزی به‌عنوان پشتیبان
محصول وارداتی و پایه تومان
شورتکد نمایش نرخ‌ها
ریست تنظیمات و بکاپ

اگر تازه شروع کرده‌اید

برای نصب، وارد کردن JSON، تست اولین بروزرسانی و تنظیم اولین محصول خارجی.

شروع سریع نصب

اگر می‌خواهید اول امکانات را یک‌جا بفهمید

برای آشنایی با بروزرسانی دستی نرخ، کرون زمان‌بندی، زبان مستقل افزونه، پاکسازی کش، Factory Reset و حذف سبک یا کامل داده‌ها.

امکانات کامل بروزرسانی دستی و کرون

اگر محصول خارجی دارید

برای فهمیدن foreign product، فرمول قیمت، پایه تومان، مرتب‌سازی صحیح قیمت و تفاوت تومان/ریال.

محصولات خارجی پیشرفته

اگر دنبال API رایگان یا منبع نرخ هستید

برای فهمیدن اینکه آیا اصلاً لازم است API بخرید، لینک آمادهٔ JSON چیست، و چه زمانی Margin یا منبع اختصاصی لازم می‌شود.

API رایگان منابع نرخ

اگر مشکل یا ابهام دارید

برای خطاهای URL، JSON، عدم اجرای Cron، عدم تغییر قیمت‌ها، یا تفاوت نرخ آزاد و رسمی.

رفع اشکال سوالات پرتکرار

اگر درباره امنیت، حذف داده‌ها یا انتقال تنظیمات نگرانید

برای بررسی درخواست‌های خارجی، داده‌های ذخیره‌شده، Export/Import، نوع حذف داده‌ها هنگام Uninstall و انتقال از سایت تست به سایت اصلی.

امنیت و حریم خصوصی بکاپ و انتقال درباره افزونه

تصویر: تب منابع نرخ

تصویر: متاباکس محصول خارجی

سوالات متداول این صفحه

جواب کوتاه به سوال‌هایی که معمولاً قبل از نصب، هنگام تنظیمات، یا زمان عیب‌یابی پیش می‌آید.

اگر تازه نصب کرده‌ام از کجا شروع کنم؟

اول شروع سریع را بخوانید، بعد نصب و منابع نرخ را تنظیم کنید. بعد از آن صفحه محصولات خارجی و در صورت نیاز شورتکد را ببینید.

اگر نمی‌خواهم API پولی بخرم کدام صفحه به درد من می‌خورد؟

صفحه API رایگان و سپس صفحه منابع نرخ را بخوانید. آن‌جا هم لینک آمادهٔ JSON را می‌بینید، هم روش پشتیبان بانک مرکزی را.

اگر فروشگاه من روی نوسان ارز حساس است کدام صفحات مهم‌ترند؟

صفحات پیشرفته، منابع نرخ و API رایگان را کنار هم بخوانید تا بفهمید Margin، Premium، Threshold و Cron را چطور تنظیم کنید.

اگر قیمت‌ها درست بروزرسانی نمی‌شوند از کجا پیگیری کنم؟

اول رفع اشکال را ببینید. اگر مشکل به منبع نرخ مربوط بود منابع نرخ را چک کنید، و اگر به قیمت محصول مربوط بود صفحه محصولات خارجی را بخوانید.
