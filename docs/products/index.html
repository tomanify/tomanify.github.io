محصولات خارجی + فرمول + پایه تومان

فقط محصولاتی که foreign=1 داشته باشند پردازش می‌شوند. ارزها allow-list هستند و ورودی‌ها validate می‌شوند.

فیلدهای لازم

  • Foreign (فعال)
  • Currency (USD/EUR/AED/TRY/CNY)
  • Price (مبلغ خارجی)
  • Margin (درصد سود)

فرمول دقیق

final_tmn_raw = price * rate_tmn * (1 + margin/100)
final_tmn     = round(final_tmn_raw / 1000) * 1000

بعد از محاسبه، مقدار نهایی به عنوان tomanify_price_base_tmn ذخیره می‌شود.

پایه تومان چه مزیتی دارد؟

برای جلوگیری از تبدیل دوباره و خطای تجمعی. تغییر واحد فقط بازنویسی قیمت ووکامرس از روی پایه است.
