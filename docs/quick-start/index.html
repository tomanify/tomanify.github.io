شروع سریع Tomanify در ۵ دقیقه

اگر می‌خواهید افزونه را سریع تست کنید و مطمئن شوید منطق کلی کار را فهمیده‌اید، همین صفحه کافی است.

۱ لینک JSON
۱ بار Update Rates
۱ محصول foreign
Margin محافظه‌کارانه

۱) لینک JSON را وارد کنید

در Tomanify → Sources، فیلد JSON URL را با این لینک یا هر API سازگار دیگر پر کنید:

https://raw.githubusercontent.com/rate-json/default/main/data.json

پیشنهاد: اگر نمی‌خواهید فعلاً درگیر روش‌های پشتیبان شوید، اول همین را تست کنید.

۲) نرخ‌ها را بروزرسانی کنید

در داشبورد روی Update Rates بزنید. اگر منبع درست باشد، آخرین زمان بروزرسانی و منبع ثبت می‌شود.

  1. شماره ۱: بعد از وارد کردن JSON یا منبع نرخ، این دکمه را بزنید تا اولین بروزرسانی انجام شود.
  2. شماره ۲: اگر خواستید بعد از تست اولیه کار به‌صورت خودکار ادامه پیدا کند، بازه زمانی اجرای کرون را همین‌جا تنظیم کنید.

۳) یک محصول را خارجی کنید

در صفحهٔ ویرایش محصول، متاباکس Tomanify را باز کنید و ارز، قیمت خرید خارجی و Margin را وارد کنید.

۴) Margin را فراموش نکنید

اگر منبع رایگان شما با تأخیر بروزرسانی می‌شود، Margin معقول می‌تواند از ضرر ناشی از نوسان کوچک بازار جلوگیری کند. برای شروع، ۳٪ تا ۵٪ برای بسیاری از فروشگاه‌ها عدد عملی‌تری است.

بعداً می‌توانید عدد دقیق‌تر را بر اساس بازار و مدل کسب‌وکار خود تنظیم کنید.

۵) نمایش نرخ‌ها به کاربر (اختیاری)

اگر می‌خواهید نرخ ارزها را هم در یک صفحه یا سایدبار نمایش دهید، این شورتکد را قرار دهید:

[tomanify_rates]

بعد از شروع سریع

منابع نرخ API رایگان محصولات خارجی پیشرفته

سوالات متداول این صفحه

جواب کوتاه به سوال‌هایی که معمولاً قبل از نصب، هنگام تنظیمات، یا زمان عیب‌یابی پیش می‌آید.

برای سریع‌ترین تست کدام روش را انتخاب کنم؟

JSON بازار آزاد سریع‌ترین و ساده‌ترین نقطه شروع است. بعد از وارد کردن URL، فقط کافی است Update Rates بزنید و یک محصول foreign را تست کنید.

اگر قیمت کالاهایم خیلی حساس است چه کنم؟

در شروع سریع هم بهتر است Margin محافظه‌کارانه مثل ۳٪ تا ۵٪ بگذارید تا اگر منبع رایگان کمی با تأخیر بروزرسانی شد، حاشیهٔ امن داشته باشید.

آیا برای شروع حتماً باید همهٔ فیدهای بانک مرکزی را هم وارد کنم؟

خیر، ولی برای پایداری بهتر توصیه می‌شود بعد از تست اولیه، فیدهای پشتیبان را هم اضافه کنید.

شورتکد از همان اول لازم است؟

نه. شورتکد فقط برای نمایش نرخ‌ها به کاربر است و بر محاسبهٔ قیمت محصولات اثری ندارد.

بعد از شروع سریع، کدام صفحه را بخوانم؟

معمولاً منابع نرخ و محصولات خارجی مهم‌ترین قدم‌های بعدی هستند؛ اگر فروشگاه شما حساس‌تر است، صفحه پیشرفته را هم بخوانید.
