شروع سریع (۵ دقیقه)

این مسیر برای راه‌اندازی اولیه است: سریع، قابل تست، و قابل توسعه.

پیشنهاد: برای شروع از JSON بازار آزاد استفاده کنید. ساده‌ترین و پایدارترین گزینه است.
یادآوری: مقادیر نرخ باید تومان باشد (عدد صحیح).

۱) JSON URL را تنظیم کنید

در Tomanify → Sources، فیلد JSON URL را با یک لینک HTTPS پر کنید.

{"values":{"USD":60000,"EUR":130000,"AED":30000,"TRY":2500,"CNY":22550}}

۲) نرخ‌ها را بروزرسانی کنید

به Dashboard بروید و روی Update Rates بزنید. باید «آخرین زمان بروزرسانی» و «منبع» آپدیت شود.

۳) یک محصول را خارجی کنید

در صفحه محصول، باکس Foreign Product (Tomanify) را باز کنید و ارز/قیمت/سود را وارد کنید. با ذخیره محصول، قیمت ووکامرس آپدیت می‌شود.

۴) ویجت نرخ‌ها را نمایش دهید (اختیاری)

[tomanify_rates]

CSS/JS فقط زمانی لود می‌شود که این شورتکد در صفحه وجود داشته باشد.
