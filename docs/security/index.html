امنیت و حریم خصوصی

Tomanify طوری طراحی شده که رفتار آن قابل پیش‌بینی و امن باشد: هیچ URL پیش‌فرضی ندارد و فقط زمانی درخواست خارجی می‌زند که شما در تنظیمات URL وارد کرده باشید.

افزونه چه چیزی را از اینترنت دریافت می‌کند؟

  • فقط نرخ ارز را از URLهایی که مدیر سایت وارد می‌کند (JSON یا RSS/XML).
  • درخواست‌ها از نوع GET هستند و هدفشان فقط دریافت فایل نرخ است.
  • URLها قبل از ذخیره و قبل از درخواست اعتبارسنجی می‌شوند و فقط HTTPS پذیرفته می‌شود.

افزونه چه چیزی را ارسال یا ردیابی می‌کند؟

  • هیچ داده‌ای از مشتریان، سفارش‌ها، پرداخت‌ها یا محتوای سایت را به بیرون ارسال نمی‌کند.
  • هیچ سیستم تبلیغاتی/آنالیتیکس/Tracking داخل افزونه وجود ندارد.

چه داده‌هایی در دیتابیس ذخیره می‌شود؟

  • Options: تنظیمات افزونه (واحد، زبان، زمان‌بندی، Source Mode، Threshold، Premium و ...)
  • Rates: نرخ‌های نهایی هر ارز (به تومان) برای استفاده در محاسبه محصولات.
  • Product Meta: برای محصولات foreign، مقادیر currency/price/margin و همچنین پایه تومان (tomanify_price_base_tmn).

حذف افزونه و پاک‌سازی داده‌ها

در حالت پیش‌فرض، هنگام حذف افزونه داده‌های محصول (metaها) حفظ می‌شوند تا اگر دوباره نصب کردید اطلاعات از بین نرود. اگر می‌خواهید همه چیز پاک شود:

  • در تب Backup گزینه Data Removal on Uninstall را فعال کنید.
  • بعد از فعال‌سازی، هنگام uninstall علاوه بر options/transients، metaهای مرتبط با محصول foreign هم حذف می‌شوند.

پیشنهادهای عملی برای امنیت بیشتر

  • اگر امکانش را دارید، فایل JSON نرخ را در یک منبع قابل اعتماد میزبانی کنید (مثلاً GitHub یا حتی هاست خودتان).
  • URLهای ناشناس/بی‌اعتبار را وارد نکنید؛ چون منبع نرخ عملاً روی قیمت‌گذاری فروشگاه اثر مستقیم دارد.
  • قبل از تغییرات مهم، از صفحه بکاپ خروجی بگیرید.

سوالات متداول این صفحه:

ما سوالات پرتکرار را که ممکن است به درک بهتر این صفحه کمک کنند را در اینجا قرار داده ایم.

آیا افزونه اطلاعات مشتریان را به بیرون می‌فرستد؟

خیر. درخواست‌های خارجی فقط برای دریافت نرخ ارز هستند و به داده‌های مشتری/سفارش ارتباطی ندارند.

چرا URL پیش‌فرض ندارد؟

برای امنیت و شفافیت. فقط URLهایی که مدیر سایت وارد می‌کند می‌توانند fetch شوند.

می‌توانم کاملاً بدون اینترنت از افزونه استفاده کنم؟

بله. Source Mode را روی Manual یا Offline بگذارید (توجه کنید Offline تخمینی است).
