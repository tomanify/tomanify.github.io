منابع نرخ

نرخ‌ها به‌صورت تومان ذخیره می‌شوند و برای محاسبه محصولات خارجی استفاده می‌گردند. پیشنهاد عمومی: JSON بازار آزاد. برای شرایط اختلال/محدودیت اینترنت: بانک مرکزی (تخمینی با Premium).

JSON بازار آزاد

این URL را در تب Sources وارد کنید. ساختار مورد انتظار: {"values":{"USD":12345,...}} (همه به تومان).

https://raw.githubusercontent.com/rate-json/default/main/data.json

RSS/XML بانک مرکزی

برای هر ارز یک URL جداگانه وارد می‌شود. سپس افزونه نرخ رسمی را می‌گیرد و با Premium به نرخ آزاد تخمینی تبدیل می‌کند.

یادآوری: نرخ آزاد = نرخ واقعی بازار. بانک مرکزی = منبع رسمی (قابل استفاده برای تخمین نرخ آزاد در زمان قطعی/محدودیت اینترنت).

Source Mode چیست و هر گزینه دقیقاً چه می‌کند؟

Source Mode تعیین می‌کند افزونه از کدام «مسیر» نرخ‌ها را بسازد. اگر روی Auto باشد، افزونه به ترتیب زیر تلاش می‌کند و اولین منبع معتبر را انتخاب می‌کند:

  • Auto: تلاش به ترتیب JSON → بانک مرکزی → آفلاین → دستی و انتخاب اولین نتیجه معتبر.
  • JSON: فقط از URL JSON بازار آزاد استفاده می‌کند.
  • CBI (Bank): فقط از RSS/XML بانک مرکزی استفاده می‌کند و با Premium نرخ آزاد را تخمین می‌زند.
  • Offline: بدون درخواست خارجی؛ با تکیه بر «مرجع ذخیره‌شده» (Reference Snapshot) و رشد سالانه (CAGR) نرخ را تخمین می‌زند.
  • Manual: بدون درخواست خارجی؛ نرخ‌ها را دقیقاً از ورودی‌های دستی شما (به تومان) می‌گیرد.

دو سناریوی پیشنهادی

  • سناریوی پیشنهادی (اکثر فروشگاه‌ها): Source Mode را روی Auto بگذارید، URL JSON را وارد کنید و برای مواقع اضطراری URLهای بانک مرکزی را هم ثبت کنید.
  • سناریوی بدون درخواست خارجی: Source Mode را روی Manual بگذارید و نرخ‌ها را دستی وارد کنید (برای محیط‌های حساس/آفلاین).

Threshold چیست و چرا مهم است؟

آستانه تغییر نرخ (Change Threshold %) مشخص می‌کند نرخ جدید «چقدر» باید با نرخ قبلی تفاوت داشته باشد تا افزونه آن را ذخیره کند و سراغ آپدیت محصولات برود. این گزینه برای کاهش نوسان، کاهش فشار روی هاست و جلوگیری از آپدیت‌های خیلی ریز طراحی شده است.

  • مثال: اگر Threshold را روی ۰٫۳٪ بگذارید، تغییرات کمتر از ۰٫۳٪ نادیده گرفته می‌شوند.
  • اگر می‌خواهید همیشه آپدیت انجام شود، مقدار را ۰ بگذارید (با احتیاط).
  • پیشنهاد عمومی: بین ۰٫۱٪ تا ۰٫۵٪ (بسته به حساسیت قیمت‌گذاری و تعداد محصولات).
نکته مهم: اگر به‌خاطر Threshold نرخ‌ها «ذخیره نشوند»، به‌صورت طبیعی آپدیت قیمت محصولات هم انجام نمی‌شود. برای جزئیات بیشتر، صفحه پیشرفته و رفع اشکال را ببینید.

بهترین منبع نرخ برای شروع چیست؟

معمولاً JSON بازار آزاد (HTTPS) ساده‌ترین و پایدارترین گزینه است.

Auto دقیقاً چه اولویتی دارد؟

در حالت Auto، افزونه به ترتیب JSON → بانک مرکزی → آفلاین → دستی تلاش می‌کند و اولین نتیجه معتبر را استفاده می‌کند.

Threshold چیست و چرا بعضی وقت‌ها آپدیت انجام نمی‌شود؟

Threshold حداقل درصد تغییر لازم برای ذخیره نرخ جدید است. اگر تغییرات کمتر از Threshold باشد، افزونه برای کاهش نوسان و فشار روی هاست، آپدیت را انجام نمی‌دهد.

فید بانک مرکزی رسمی است؛ چطور به بازار آزاد نزدیک می‌شود؟

با ضریب Premium (نسبت بازار آزاد به رسمی). افزونه نرخ رسمی را می‌گیرد و با Premium به نرخ آزاد تخمینی تبدیل می‌کند.

می‌توانم هیچ درخواست خارجی نزنم؟

بله. Source Mode را روی Manual بگذارید و نرخ‌ها را دستی به تومان وارد کنید.

چرا URLها به‌صورت پیش‌فرض خالی هستند؟

به‌خاطر امنیت. افزونه هیچ URL پیش‌فرضی قرار نمی‌دهد و فقط URLهایی را که شما وارد می‌کنید fetch می‌کند.
