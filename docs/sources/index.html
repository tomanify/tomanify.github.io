منابع نرخ در Tomanify

مهم‌ترین صفحه برای راه‌اندازی درست افزونه همین‌جاست؛ چون کیفیت منبع نرخ تعیین می‌کند قیمت‌های فروشگاه چقدر دقیق، پایدار و قابل اتکا باشند.

برای رعایت امنیت و الزامات وردپرس، افزونه هیچ URL آماده‌ای را از قبل در تنظیمات قرار نمی‌دهد. یعنی شما باید لینک‌های منبع را خودتان وارد کنید.

۱) JSON بازار آزاد — بهترین راه برای شروع

اگر می‌خواهید نرخ واقعی‌تری نسبت به نرخ‌های رسمی داشته باشید و بدون هزینهٔ API کار را شروع کنید، JSON بازار آزاد مناسب‌ترین گزینه است.

https://raw.githubusercontent.com/rate-json/default/main/data.json

اگر JSON اختصاصی دارید، کافی است خروجی را به فرمت سازگار با افزونه برگردانید.

۲) RSS/XML بانک مرکزی — روش پشتیبان

اگر به هر دلیل منبع آزاد در دسترس نبود، فیدهای رسمی بانک مرکزی می‌توانند لایهٔ پشتیبان شما باشند. با کمک Premium، نرخ رسمی به نرخ آزاد نزدیک‌تر می‌شود.

ساختار JSON سازگار

اگر خودتان API دارید یا برایتان API سفارشی ساخته می‌شود، ساده‌ترین فرمت سازگار این است:

{"values":{"USD":65000,"EUR":71000,"AED":17800,"TRY":2300,"CNY":9000}}

Source Mode را چطور انتخاب کنم؟

  • JSON / Auto: برای منبع اصلی بازار آزاد
  • Bank: وقتی می‌خواهید روی فیدهای رسمی تکیه کنید
  • Offline: برای شرایط اضطراری و عدم دسترسی به منابع بیرونی
  • Manual: وقتی همه چیز را دستی کنترل می‌کنید

تصویر: تب منابع نرخ

  1. شماره ۱: این فیلد برای آدرس فایل JSON بازار آزاد است. باید یک لینک HTTPS وارد کنید که خروجی آن با ساختار سازگار افزونه و با مقادیر تومانی برگردد.
  2. شماره‌های ۲ تا ۶: این ردیف‌ها برای وارد کردن لینک RSS/XML رسمی هر ارز از بانک مرکزی هستند. هر ارز فیلد جداگانه دارد و لازم نیست همه را یک‌جا پر کنید.
  3. شماره ۷: این بخش مربوط به نرخ‌های دستی است. اگر Source Mode روی Manual باشد، نرخ هر ارز را در همین فیلدها وارد می‌کنید.

چه ترکیبی برای بیشتر فروشگاه‌ها خوب است؟

  1. JSON بازار آزاد به‌عنوان منبع اصلی
  2. فیدهای بانک مرکزی به‌عنوان پشتیبان
  3. Threshold معقول برای جلوگیری از نویز
  4. Margin محافظه‌کارانه برای پوشش تأخیر احتمالی
  5. Offline یا Manual برای شرایط اضطراری

سوالات متداول این صفحه

جواب کوتاه به سوال‌هایی که معمولاً قبل از نصب، هنگام تنظیمات، یا زمان عیب‌یابی پیش می‌آید.

بهترین Source Mode برای شروع کدام است؟

برای بیشتر فروشگاه‌ها، JSON بازار آزاد بهترین نقطه شروع است. بعد از آن فیدهای بانک مرکزی را به‌عنوان لایهٔ پشتیبان اضافه کنید.

Premium دقیقاً چه مشکلی را حل می‌کند؟

Premium کمک می‌کند نرخ رسمی بانک مرکزی به نرخ آزاد نزدیک شود. یعنی وقتی از RSS/XML رسمی استفاده می‌کنید، تبدیل صرفاً با همان رقم رسمی باقی نمی‌ماند.

اگر JSON من کار نکرد اول کجا را چک کنم؟

HTTPS بودن URL، ساختار JSON، عددی بودن مقادیر، و این‌که آیا واقعاً همه ارزها به تومان برمی‌گردند یا نه.

آیا می‌توانم فقط بعضی ارزها را از بانک مرکزی بگیرم؟

بله. هر فید برای هر ارز جداگانه وارد می‌شود و می‌توانید ترکیب منبع‌ها را مطابق نیازتان بچینید.

Offline Mode چه زمانی مفید است؟

وقتی هیچ منبع بیرونی در دسترس نیست و می‌خواهید افزونه با مدل مرجع/پیش‌بینی، کار را به‌طور موقت ادامه دهد.

Threshold چرا گاهی باعث می‌شود قیمت‌ها آپدیت نشوند؟

چون تغییرهای خیلی کوچک را برای جلوگیری از نویز نادیده می‌گیرد. اگر می‌خواهید حساس‌تر باشید، مقدار آن را کمتر کنید.
