منابع نرخ (JSON / RSS/XML / دستی / آفلاین)

هرچه منبع نرخ پایدارتر باشد، قیمت‌گذاری فروشگاه شما قابل پیش‌بینی‌تر می‌شود. Tomanify خروجی نهایی را به صورت عدد صحیح تومان ذخیره می‌کند.

JSON بازار آزاد (پیشنهادی)

خروجی باید شبیه مثال زیر باشد و مقادیر تومان باشد:

{"values":{"USD":60000,"EUR":130000,"AED":30000,"TRY":2500,"CNY":22550}}
امنیت: URL باید HTTPS باشد. افزونه هیچ URL پیش‌فرضی قرار نمی‌دهد.

RSS/XML رسمی + Premium

برای نرخ‌های رسمی، معمولاً به ضریب Premium نیاز دارید تا به نرخ نزدیک بازار آزاد برسید.

Premium یعنی نسبت «بازار آزاد / رسمی». عدد بزرگ‌تر → نرخ آزاد تخمینی بیشتر.

نرخ دستی (Manual)

اگر می‌خواهید هیچ درخواست خارجی نزنید، بهترین گزینه Manual است. برای هر ارز عدد تومان وارد کنید.

مدل آفلاین (Offline)

برای زمانی که منبع بیرونی ندارید یا موقتاً قطع است. نرخ‌ها با الگوریتم داخلی تخمین زده می‌شوند.

Change threshold % (کنترل حساسیت تغییر)

اگر تغییر نرخ کمتر از آستانه باشد، نرخ قبلی نگه داشته می‌شود تا قیمت‌ها بی‌دلیل تکان نخورند.
