رفع اشکال

اگر چیزی درست کار نمی‌کند، از این چک‌لیست‌ها شروع کنید. اکثر مشکلات در کمتر از ۲ دقیقه پیدا می‌شوند.

۱) قیمت محصولات آپدیت نمی‌شود

  • آیا محصول واقعاً foreign است؟ (در متاباکس Tomanify تیک خورده باشد)
  • آیا Currency انتخاب شده و Price (قیمت خارجی) وارد شده است؟
  • آیا نرخ آن ارز در Dashboard نمایش داده می‌شود؟
  • اگر نرخ‌ها «خیلی کم» تغییر کرده‌اند، Threshold ممکن است باعث شود افزونه نرخ جدید را ذخیره نکند.
  • اگر سایت کش دارد، بعد از بروزرسانی کش را پاک کنید.

۲) نرخ‌ها آپدیت می‌شوند ولی قیمت محصول تغییر نمی‌کند

این حالت معمولاً یکی از این علت‌ها را دارد:

  • Threshold: نرخ جدید ذخیره نشده، پس آپدیت محصولات هم اجرا نشده است.
  • عدم تطابق ارز: محصول روی EUR است اما شما فقط USD را نرخ داده‌اید.
  • داده محصول ناقص: price/currency/margin درست ذخیره نشده‌اند.

راهنما: صفحه پیشرفته (Threshold) و محصولات خارجی را ببینید.

۳) خطاهای URL / HTTPS / SSL

  • افزونه فقط URLهای HTTPS را قبول می‌کند. (به‌خاطر امنیت)
  • اگر هاست/شبکه شما به دامنه منبع دسترسی ندارد، منبع را عوض کنید یا فایل نرخ را در هاست خودتان میزبانی کنید.
  • برای JSON مطمئن شوید خروجی واقعاً JSON است و ساختار values دارد.

۴) فرمت JSON اشتباه است

نمونه ساختار صحیح (مقادیر به تومان):

{
  "values": {
    "USD": 12345,
    "EUR": 13456,
    "AED": 3456,
    "TRY": 456,
    "CNY": 1789
  }
}

فقط همین کلیدها خوانده می‌شوند. اگر نام ارز یا ساختار متفاوت باشد، افزونه آن را نادیده می‌گیرد.

۵) ویجت نمایش داده نمی‌شود

  • شورتکد [tomanify_rates] را در یک برگه/پست قرار دهید.
  • اگر از کش/مینیفای استفاده می‌کنید، کش را پاک کنید. (اسکریپت/استایل فقط وقتی شورتکد وجود داشته باشد لود می‌شوند.)

۶) کران اجرا نمی‌شود

  • WP-Cron به بازدید وابسته است. اگر سایت بازدید کم دارد، ممکن است دیر اجرا شود.
  • برای دقت بالا، از کران واقعی هاست یا یک سرویس ping دوره‌ای برای اجرای wp-cron.php استفاده کنید.

۷) فرق «آپدیت نرخ» با «آپدیت قیمت محصولات» چیست؟

  • Update Rates: نرخ‌ها را از منبع می‌گیرد و ذخیره می‌کند. اگر تغییر از Threshold کمتر باشد، هیچ ذخیره‌ای انجام نمی‌شود.
  • آپدیت محصولات: وقتی نرخ ذخیره شد، محصولات foreign مرتبط با همان ارز آپدیت می‌شوند. همچنین با ذخیره هر محصول، یک محاسبه تک‌محصولی انجام می‌شود.

سوالات متداول این صفحه:

ما سوالات پرتکرار را که ممکن است به درک بهتر این صفحه کمک کنند را در اینجا قرار داده ایم.

نرخ‌ها آپدیت می‌شوند ولی قیمت محصول تغییر نمی‌کند

مطمئن شوید محصول foreign است و ارز/قیمت/سود دارد؛ سپس Threshold و کش سایت را بررسی کنید.

چرا کران اجرا نمی‌شود؟

WP-Cron وابسته به بازدید است؛ پیشنهاد: کران واقعی هاست یا ping دوره‌ای.

فرمت JSON دقیقاً باید چطور باشد؟

یک شیء JSON با کلید values که نرخ‌ها را به تومان نگه می‌دارد. نمونه در همین صفحه آمده است.

پرج کش چه محدودیتی دارد؟

فقط URLهای هم‌دامنه purge می‌شوند تا ریسک امنیتی کاهش یابد.
