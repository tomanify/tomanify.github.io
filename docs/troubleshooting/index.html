رفع اشکال سریع

چک‌لیست‌های کوتاه برای رایج‌ترین مشکلات در نرخ‌ها، محصولات خارجی، کران و کش.

نرخ‌ها آپدیت می‌شوند ولی قیمت محصولات تغییر نمی‌کند

محصول باید foreign باشد و ارز/قیمت/سود داشته باشد. سپس threshold را بررسی کنید. اگر نرخ ارز مربوطه صفر/ناموجود باشد، پردازش نمی‌شود.

فقط می‌خواهم نرخ‌ها دستی باشد

Source Mode را روی Manual بگذارید و برای هر ارز عدد تومان وارد کنید. URLها را خالی بگذارید.

کران اجرا نمی‌شود یا دیر اجرا می‌شود

در هاست‌های اشتراکی، WP-Cron وابسته به بازدید است. می‌توانید با کران واقعی هاست یا سرویس‌های Cron یک ping به سایت بزنید.

پرج (Purge) کش

پیش‌فرض خاموش است. اگر روشن شود فقط URLهای هم‌دامنه purge می‌شوند تا ریسک امنیتی کاهش یابد.
