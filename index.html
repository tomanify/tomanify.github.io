خانه

تومنی‌فای؛ راه‌حل واقعی برای تبدیل قیمت ارز خارجی به تومان/ریال در ووکامرس

Tomanify برای فروشگاه‌هایی ساخته شده که قیمت خریدشان دلاری، یورویی، درهمی، لیر یا یوانی است اما باید قیمت نهایی را به تومان/ریال و با منطق واقعی بازار ایران در ووکامرس ذخیره کنند.

بروزرسانی خودکار قیمت‌ها
بدون نیاز اجباری به API پولی
پایه تومان برای جلوگیری از تبدیل دوباره
مرتب‌سازی و فیلتر قیمت به‌هم نمی‌ریزد
پاک‌سازی هوشمند کش همان صفحه
زبان مستقل افزونه و حذف قابل کنترل

اگر بپرسی «چگونه قیمت‌های سایتم را خودکار کنم؟»

جواب کوتاه این است: برای هر محصول خارجی مبلغ ارزی و نوع ارز را ثبت می‌کنی، منبع نرخ را مشخص می‌کنی، و Tomanify قیمت نهایی ووکامرس را با فرمول واقعی بازار ایران می‌سازد و بروزرسانی می‌کند.

راهنمای بروزرسانی خودکار

اگر بپرسی «آیا می‌شود قیمت فروشگاه اینترنتی را اتوماتیک تغییر داد؟»

بله. این افزونه فقط نرخ ارز را نشان نمی‌دهد؛ خودِ قیمت ووکامرس را هم بازنویسی می‌کند. برای همین سورت، فیلتر و گزارش‌گیری قیمت هم به‌هم نمی‌ریزد.

محصولات خارجی

اگر API پولی نخواهم چه؟

در بیشتر سناریوها لازم نیست از سرویس‌دهنده‌های نرخ ارز اشتراک ماهانه بخرید. می‌توانید از فید رایگان بازار آزاد استفاده کنید، فیدهای بانک مرکزی را به‌عنوان پشتیبان بگذارید، و برای کنترل ریسک نوسان لحظه‌ای یک Margin معقول تعریف کنید.

API رایگان و سناریوهای جایگزین

چرا قیمت‌ها فقط ظاهری عوض نمی‌شوند؟

Tomanify صرفاً یک نرخ را روی ظاهر سایت نمایش نمی‌دهد. افزونه، قیمت نهایی ووکامرس را هم از روی «پایه تومان» بازنویسی می‌کند؛ بنابراین مرتب‌سازی از ارزان به گران، فیلتر قیمت، و استفادهٔ سایر افزونه‌ها از فیلد قیمت واقعی ووکامرس درست می‌ماند.

مرور کامل قابلیت‌ها

پاک‌سازی کش بدون افزونه اضافی

اگر قیمت محصول عوض شد اما به‌خاطر کش کاربر هنوز قیمت قدیمی را می‌بیند، می‌توانید Purge اختیاری را فعال کنید تا فقط همان صفحه‌ها دوباره تازه شوند؛ بدون این‌که لازم باشد کل کش سایت را خالی کنید.

پاک‌سازی کش

زبان افزونه جدا از زبان کل سایت

ممکن است پنل وردپرس شما انگلیسی باشد اما بخواهید فقط Tomanify فارسی باشد؛ یا برعکس. زبان داخلی افزونه مستقل است و همین موضوع کار با آن را برای تیم‌های مختلف راحت‌تر می‌کند.

زبان افزونه

ریست تنظیمات و حذف سبک یا کامل

اگر پیکربندی‌ها به‌هم بریزد، می‌توانید تنظیمات را به حالت پیش‌فرض برگردانید. هنگام حذف هم می‌توانید تصمیم بگیرید همه داده‌ها کامل پاک شوند یا داده‌های مهم محصولات خارجی برای استفاده بعدی باقی بمانند.

ریست و حذف

وقتی منبع اصلی در دسترس نیست

اگر JSON بازار آزاد قطع شود یا اینترنت محدود شود، می‌توانید فیدهای RSS/XML بانک مرکزی را فعال نگه دارید. اگر باز هم لازم باشد، Offline Mode و نرخ دستی آخرین لایه‌های پشتیبان هستند.

از کجا شروع کنم؟

اگر اولین بار است که Tomanify را نصب می‌کنید، بهترین مسیر این است:

شروع سریع API رایگان قابلیت‌ها بروزرسانی خودکار رفع اشکال

عبارت‌هایی که کاربران واقعاً جستجو می‌کنند

این سایت فقط برای اسم افزونه ساخته نشده. اگر کسی عباراتی مثل «چگونه قیمت‌های سایتم را خودکار کنم؟»، «آموزش بروزرسانی خودکار قیمت‌های وردپرس»، «تبدیل خودکار قیمت دلار به تومان در ووکامرس»، «راه‌حل تغییر اتوماتیک قیمت اجناس فروشگاه اینترنتی» یا حتی نام‌های مختلف مثل «تومنی فای»، «تومانی فای» و «تومانیفای» را جستجو کند، این مستندات باید جواب روشن و عملی بدهند.

سوالات متداول این صفحه

جواب کوتاه به سوال‌هایی که معمولاً قبل از نصب، هنگام تنظیمات، یا زمان عیب‌یابی پیش می‌آید.

Tomanify دقیقاً چه کاری انجام می‌دهد؟

قیمت خرید خارجی محصولات را با نرخ ارز به تومان یا ریال تبدیل می‌کند، قیمت نهایی ووکامرس را بازنویسی می‌کند، و بعداً هم می‌تواند آن را به‌صورت زمان‌بندی‌شده بروزرسانی کند.

آیا برای شروع حتماً باید API پولی بخرم؟

برای اکثر فروشگاه‌ها خیر. می‌توانید از JSON رایگان بازار آزاد استفاده کنید و برای پوشش ریسک نوسان، Margin معقول تعریف کنید.

آیا می‌شود قیمت‌های فروشگاه اینترنتی را خودکار تغییر داد؟

بله. اگر محصولات خارجی را با مبلغ ارزی تعریف کنید و منبع نرخ را مشخص کنید، Tomanify می‌تواند قیمت نهایی ووکامرس را به‌صورت دستی یا زمان‌بندی‌شده بروزرسانی کند.

اگر زبان وردپرس انگلیسی باشد ولی بخواهم خود افزونه فارسی باشد چه؟

زبان داخلی Tomanify مستقل از زبان کلی سایت است و می‌توانید فقط زبان افزونه را تغییر دهید.

اگر بعد از تغییر قیمت، کاربر هنوز قیمت قدیمی ببیند چه می‌شود؟

می‌توانید قابلیت purge را فعال کنید تا کش همان صفحه‌ها پس از بروزرسانی قیمت دوباره پاک‌سازی شود.

هنگام حذف افزونه همه داده‌ها پاک می‌شوند؟

این موضوع قابل تنظیم است؛ می‌توانید حذف کامل داشته باشید یا حذف سبک تا داده‌های مهم محصولات خارجی برای استفاده بعدی باقی بمانند.
