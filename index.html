خانه

تومنی‌فای؛ راه‌حل واقعی برای قیمت‌گذاری محصولات خارجی در ووکامرس

Tomanify برای فروشگاه‌هایی ساخته شده که قیمت خریدشان دلاری، یورویی، درهمی، لیر یا یوانی است اما باید قیمت نهایی را به تومان/ریال و با منطق واقعی بازار ایران در ووکامرس ذخیره کنند.

نرخ بازار آزاد + روش‌های پشتیبان
بدون نیاز اجباری به API پولی
پایه تومان برای جلوگیری از تبدیل دوباره
مرتب‌سازی و فیلتر قیمت به‌هم نمی‌ریزد
صف‌بندی و بروزرسانی خودکار محصولات
سازگار با هاست اشتراکی و افزونه‌های کش

چرا این سایت ساخته شده است؟

هدف این سایت این است که همه چیز را یک‌جا و به زبان فارسی توضیح بدهد: از «Tomanify چیست؟» تا «منبع نرخ را دقیقاً از کجا بگیرم؟» و «اگر فروشگاه من حساس است چه تنظیمی بهتر است؟». یعنی فقط مستندات فنی نباشد، بلکه برای صاحب فروشگاه هم قابل استفاده باشد.

در عمل این سایت، هم مرجع راه‌اندازی است، هم مرجع تصمیم‌گیری برای انتخاب منبع نرخ، Margin، Cron و استراتژی قیمت‌گذاری.

برای چه فروشگاه‌هایی مناسب است؟

اگر قیمت خرید شما با ارز خارجی ثبت می‌شود و نمی‌خواهید هر بار دستی قیمت کالاها را عوض کنید، Tomanify برای شما ساخته شده است. این سناریو مخصوصاً برای فروشگاه‌های محصولات وارداتی، گجت، قطعات، مد، لوازم جانبی و کالاهای وابسته به نرخ ارز مهم است.

هم برای فروشگاه‌های کوچک مناسب است، هم برای فروشگاه‌هایی که تعداد محصولاتشان زیاد است و نیاز به بروزرسانی صف‌بندی‌شده دارند.

لینک آمادهٔ JSON بازار آزاد

برای راه‌اندازی سریع، ساده‌ترین گزینه این است که در تب Sources لینک JSON بازار آزاد را وارد کنید. این روش برای اکثر فروشگاه‌ها بهترین نقطه شروع است.

https://raw.githubusercontent.com/rate-json/default/main/data.json

بعد از ذخیره، در داشبورد روی Update Rates بزنید و سپس یک محصول را به‌عنوان foreign تنظیم کنید.

اگر API پولی نخواهم چه؟

در بیشتر سناریوها لازم نیست از سرویس‌دهنده‌های نرخ ارز اشتراک ماهانه بخرید. می‌توانید از فید رایگان بازار آزاد استفاده کنید، فیدهای بانک مرکزی را به‌عنوان پشتیبان بگذارید، و برای کنترل ریسک نوسان لحظه‌ای یک Margin معقول مثل ۳٪ تا ۵٪ تعریف کنید.

برای فروشگاه‌هایی که به نوسان‌های بسیار کوچک هم حساس‌اند، API شخصی یا اختصاصی انتخاب بهتری است.

چرا قیمت‌ها فقط ظاهری عوض نمی‌شوند؟

Tomanify صرفاً یک نرخ را روی ظاهر سایت نمایش نمی‌دهد. افزونه، قیمت نهایی ووکامرس را هم از روی «پایه تومان» بازنویسی می‌کند؛ بنابراین مرتب‌سازی از ارزان به گران، فیلتر قیمت، و استفادهٔ سایر افزونه‌ها از فیلد قیمت واقعی ووکامرس درست می‌ماند.

این تفاوت مهمی است که باعث می‌شود فروشگاه فقط «نمایشی» ارزی نباشد، بلکه واقعاً با قیمت نهایی صحیح کار کند.

وقتی منبع اصلی در دسترس نیست

اگر JSON بازار آزاد قطع شود یا اینترنت محدود شود، می‌توانید فیدهای RSS/XML بانک مرکزی را فعال نگه دارید. افزونه با کمک Premium نرخ رسمی را به نرخ آزاد نزدیک می‌کند. اگر باز هم لازم باشد، Offline Mode و نرخ دستی آخرین لایه‌های پشتیبان هستند.

یعنی سایت شما فقط به یک لینک وابسته نمی‌ماند.

از کجا شروع کنم؟

اگر اولین بار است که Tomanify را نصب می‌کنید، بهترین مسیر این است:

شروع سریع API رایگان منابع نرخ محصولات خارجی

جواب کوتاه به سوال‌هایی که معمولاً قبل از نصب، هنگام تنظیمات، یا زمان عیب‌یابی پیش می‌آید.

Tomanify دقیقاً چه کاری انجام می‌دهد؟

قیمت خرید خارجی محصولات را با نرخ ارز به تومان یا ریال تبدیل می‌کند، قیمت نهایی ووکامرس را بازنویسی می‌کند، و بعداً هم می‌تواند آن را به‌صورت زمان‌بندی‌شده بروزرسانی کند.

آیا برای شروع حتماً باید API پولی بخرم؟

برای اکثر فروشگاه‌ها خیر. می‌توانید از JSON رایگان بازار آزاد استفاده کنید و برای پوشش ریسک نوسان، Margin معقول تعریف کنید.

اگر اینترنت محدود شود یا JSON در دسترس نباشد چه می‌شود؟

می‌توانید فیدهای RSS/XML بانک مرکزی را به‌عنوان روش پشتیبان تنظیم کنید و در صورت لزوم از Offline Mode یا نرخ دستی هم استفاده کنید.

چرا سورت محصولات از ارزان به گران درست می‌ماند؟

چون افزونه فقط نرخ نمایشی را عوض نمی‌کند؛ قیمت نهایی ووکامرس را هم از روی پایه تومان بازنویسی می‌کند تا دیتابیس بر اساس مبلغ واقعی مرتب‌سازی شود.

آیا افزونه به‌صورت پیش‌فرض URL منبع نرخ دارد؟

خیر. برای رعایت امنیت و الزامات وردپرس، فیلدهای URL به‌صورت پیش‌فرض خالی هستند و فقط بعد از وارد کردن آدرس توسط مدیر، درخواست خارجی انجام می‌شود.

اگر فروشگاه من روی نوسان‌های خیلی کوچک حساس باشد چه کنم؟

در این حالت یا Margin محافظه‌کارانه بگذارید، یا از API شخصی/سفارشی‌تر استفاده کنید. منبع رایگان برای اکثر فروشگاه‌ها مناسب است ولی لزوماً برای سناریوهای بسیار حساس کافی نیست.
